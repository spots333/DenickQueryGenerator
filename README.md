/denick command generator for my Duels Denicker bot.
To use:
After building/downloading, place it in its own folder for simplicity and run it. It will ask for your mc logs file, that depends on which client you are using. Currently it works for at least Lunar and BLC. Search up where they are held on google.
When giving it the file path, make sure to include the latest.log file at the end, so something like this if you are using Lunar: "C:\Users\YourUser\\.lunarclient\offline\multiver\logs\latest.log"
If you change clients or paths, just delete the spotsDenickerLogFile.txt file, or manually update it in the generated spotsDenickerLogFile.txt.
Run it alongside your client.
